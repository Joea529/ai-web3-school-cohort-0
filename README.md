# AI × Web3 School — 庄龙的学习仓库

> **AI × Web3 School** 是一个面向 builders 的开源学习计划。把 AI 和 Web3 真正交叉时的问题拆清楚：模型能力、Agent 工作流、工具调用、钱包、签名、支付、身份、权限、安全执行、治理协作和可验证记录。
